# LaQoo
基于Predidit/Kazumi的番剧APP，支持流媒体在线观看，支持弹幕。


## 支持平台

- Android 10 及以上
- Windows 10 及以上
- MacOS 10.15 及以上
- Linux (实验性)
- iOS (需要自签名)

。
